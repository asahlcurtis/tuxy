Tuxy A virtual assistant.

Tuxy is a virtual assistant inspired by Clippy to automate tasks, customize your os, and general help of office work. It is pre-alpha but updates will come every two weeks give or take. Help Wanted.

Features:


Project manager/task manager taskman (WIP)
Flatpak manager Flattux (WIP)
Text editor T-Edit (kinda WIP mostly finished)
Custom assistants

Philosophy of tuxy:


Tuxy is a virtual assistant, but a virtual assistant with a lot of upcoming features. Tuxy is also very easy to customize. Tuxy is built to be edited, it loads things like assistants so you can add your own assistant images, It has plenty of comments and documentation to easily learn how it works like the back of your hand. Tuxy also introduces libraries like lib/node.h which is a header file that creates a dragable object, and this isn't the only custom object tuxy will create, I plan to make many different classes to add functionality for the software, and since tuxy is MIT licensed you can use these header files, even commericaly.
